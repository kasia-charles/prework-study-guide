HTML

  • HTML creates structure to a webpage and is made up of elements that work together to provide a framework.
  • Element types include text, image, hader, list.
  • The head element contains informaiton about the webpage.
  • The body element represents the visible content shown to the user.
  • Element refers to the entire element from opening tag to closing tag.
  • Tag refers only to what appears inside the angled brackets.

CSS

  • CSS stands for Cascading Style Sheets
  • CSS adds style to a webpage and defines how webpage content is presented to users
  • 3 ways to style a webpage using CSS are inline CSS, internal CSS style sheet, external CSS style sheet
  • Anytime you create an external CSS file, you must link it to the HTML file

Git

  • git status: checks what branch we are currently on.
  • git checkout -b branch-name: creates a new branch and switches to it.
  • git add -A: use this to stage your cahnges and commit all files in teh working branch
  • git commit -m "______": use the m to assocaite a message with our commit, ensure it is a short descriptive message of the changes implemented
  • git pull origin main: this pulls the base branch into our feature & it is best to do this before you push up any code
  • git push origin ______: this pushes up all of our changes when you name the branch you are pushing with this command

JavaScript

  • JavaScript (JS) is a high-level dynamic programming language that allows users to interact with websites as it adds functionality to your webpage
  • A variable is a named container that allows us to store data in our code.
  • console.log method is used to output a message to the web console by adding an argument
  • Control flow is the order in which a computer executes code in a script
  • Functions=a set of instructions that tells the computer how to perform a certain task
  • 2 requirements for a JS function: 1- must define the function and 2- must call the function.
